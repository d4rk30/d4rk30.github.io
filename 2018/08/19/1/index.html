Ubuntu系统配置指南

发表于 2018-08-19 更新于 2019-07-19

快速配置Ubuntu系统，所有配置都在Ubuntu-16.04LTS下完成。

0x00 配置root登录

1.修改root密码

root用户默认是没有密码的，因此也就无法使用。想用root的话，就得给root用户设置一个密码。

输入密码即可。

2.修改ssh配置

将PermitRootLogin这项的值将其改为yes，然后保存退出并重启ssh服务既可。

最后验证root权限是否可以远程登录。

0x02 安装PHP及环境配置

1.安装PHP

2.设置fpm日志

一般在安装php时候已经会自动安装fpm，如果没有安装，先去安装fpm。

fpm日志路径：/etc/php/7.0/fpm/pool.d/www.conf
备注：根据不同php版本，有可能不是7.0，需要按照实际情况进行查看。

找到;access.log = log/$pool.access.log改成access.log = /var/log/php.access.log，然后重启服务。

3.其他关键配置

user = www-data
group = www-data
listen = /run/php/php7.0-fpm.sock
